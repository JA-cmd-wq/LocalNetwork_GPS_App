# LoRa GPS App 截屏指南

> 建议按以下顺序截取，覆盖 APP 所有核心功能界面。

---

## 1. 启动画面 (SplashScreen)

**文件**: `src/components/SplashScreen.tsx`
**操作**: 打开 APP，在动画播放中截屏
**要点**: 粒子动画 + Logo 旋转 + 进度条

---

## 2. 首次设置页 — API Key 输入

**文件**: `src/App.tsx` (第 124~191 行)
**操作**: 清除 localStorage 后打开 APP（或首次安装）
**要点**: 毛玻璃卡片、Key + 安全密钥两个输入框、渐变按钮

---

## 3. 主界面 — 地图 + 侧边栏 (桌面端)

**文件**: `src/App.tsx` + `src/components/MapView.tsx`
**操作**: 输入 Key 后进入主界面，展开侧边栏
**要点**:
- 顶部 Header（毛玻璃效果、连接按钮、模式切换）
- 左侧侧边栏（设备列表 + 告警面板）
- 右侧高德地图（标记点、设备标签）

---

## 4. 主界面 — 手机端响应式

**操作**: 浏览器 F12 → 切换到手机视图 (375×812)，或用真机
**要点**:
- 汉堡菜单按钮
- 侧边栏覆盖抽屉模式
- 紧凑 Header

---

## 5. 设备面板 (DevicePanel)

**文件**: `src/components/DevicePanel.tsx`
**操作**: 连接设备后或使用模拟数据，展开侧边栏
**要点**:
- 设备卡片（头像、名称、坐标、POI 地名）
- 信号强度 (RSSI/SNR)
- "搜星中…" 状态（黄色文字）
- 选中设备高亮效果

---

## 6. 连接面板 (ConnectionPanel)

**文件**: `src/components/ConnectionPanel.tsx`
**操作**: 点击 Header 中的「连接」按钮，展开下拉
**要点**:
- 未连接状态 → 蓝牙 SPP / USB 串口两个选项
- 已连接状态 → 绿色「已连接」+ 设备名 + 断开按钮
- 连接中 → 旋转 Loading 图标

**建议截 3 张**: 未连接 / 连接中 / 已连接

---

## 7. 告警面板 (AlertPanel)

**文件**: `src/components/AlertPanel.tsx`
**操作**: 侧边栏底部「告警」区域
**要点**:
- 点击「添加」→ 展开表单（类型选择、设备选择、参数）
- 已有规则列表（开/关切换、删除按钮）
- 触发中的告警（红色脉冲图标）

**建议截 2 张**: 添加表单 / 规则列表

---

## 8. 地图标注细节

**文件**: `src/components/MapView.tsx`
**操作**: 有多台设备在线时，放大地图
**要点**:
- 设备气泡标记（圆形头像 + 脉冲动画）
- 设备标签（毛玻璃背景、SF Pro 字体）
- 设备间距离连线 + 距离标签
- 轨迹线（开启轨迹显示后）

---

## 9. 工具栏 (Toolbar)

**文件**: `src/components/Toolbar.tsx`
**操作**: 桌面端侧边栏顶部 / 手机端侧边栏内
**要点**: 轨迹开关、导出 JSON、导出 CSV 三个按钮

---

## 10. 设置弹窗

**文件**: `src/App.tsx` (第 430~490 行)
**操作**: 点击 Header 齿轮图标
**要点**: Key + 安全密钥编辑、保存并刷新

---

## 11. 实时连接 Toast

**操作**: 蓝牙连接成功后，地图顶部出现绿色浮动提示
**要点**: "实时模式 — [设备名]" + 脉冲圆点

---

## 截屏技巧

- **浏览器**: `npm run dev` 后访问 localhost:5173，按 F12 可切手机视图
- **手机**: 安装 APK 后直接截屏
- **毛玻璃效果**: 确保地图已加载，背景有内容才能看到磨砂效果
- **模拟数据**: 不连接设备时 APP 会自动生成模拟数据，可用于截屏
- **深色主题**: 截屏时确保系统为深色模式，效果最佳
